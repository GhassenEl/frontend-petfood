# FitClub — Gestion salle de sport

Mini-projet web **autonome** : tableau de bord pour une salle de fitness (adhérents, planning, abonnements).

## Fonctionnalités démo

- Tableau de bord avec KPI
- Liste adhérents avec recherche
- Planning hebdomadaire des cours
- Grille des formules d'abonnement

## Lancer

Ouvrir `index.html` dans Chrome/Firefox, ou :

```bash
npx --yes serve -l 5501
```

Stack : HTML5, CSS3, JavaScript vanilla — aucune dépendance.
