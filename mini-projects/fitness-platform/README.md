# FitConnect — Plateforme fitness en ligne

Mini-projet web **autonome** : plateforme de coaching fitness (programmes, coachs, réservation de séances).

## Fonctionnalités démo

- Page d'accueil marketing
- Catalogue de programmes avec filtres
- Profils coachs
- Formulaire de réservation avec confirmation

## Lancer

Ouvrir `index.html` dans le navigateur, ou :

```bash
npx --yes serve -l 5502
```

Stack : HTML5, CSS3, JavaScript vanilla — aucune dépendance, aucun lien avec PetfoodTN.
