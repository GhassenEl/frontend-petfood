# DriveRent — Location de voitures

Mini-projet autonome : flotte, clients, locations et réservations.

## Fonctionnalités

- Gestion flotte (ajout / suppression véhicules)
- Clients avec recherche
- Locations dynamiques (statuts cliquables)
- KPIs en temps réel · `localStorage`
- Thème noir & blanc

## Lancer

**Web :**
```bash
npx --yes serve -l 5525
```

**Flutter :**
```bash
cd ../flutter/driverent_location
flutter run -d web-server --web-port=5525
```
