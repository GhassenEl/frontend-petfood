# HRHub — Plateforme RH

App Flutter mobile/web : gestion des ressources humaines.

## Fonctionnalités

- **Dashboard** : employés actifs, masse salariale, congés en attente, postes ouverts
- **Employés** : fiches, départements, salaires, statuts cliquables
- **Congés** : demandes avec validation (En attente → Approuvé → Refusé)
- **Recrutement** : offres d'emploi, candidats, avancement des processus
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5532
```

Ouvrir http://localhost:5532
