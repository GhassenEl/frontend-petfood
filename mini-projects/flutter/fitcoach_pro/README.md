# FitCoach — Coach sportif

Application Flutter de coaching sportif avec IA, calories, nutrition, exercices et montre connectée.

## Fonctionnalités

- **Dashboard** — bilan calories, pas, FC, exercices
- **Exercices** — séances, validation, calories brûlées
- **Nutrition** — repas, macros (protéines, glucides, lipides)
- **Montre / capteurs** — sync FitBand Pro (pas, FC, SpO₂, minutes actives)
- **CoachBot IA** — calories, nutrition, plan séance, récupération
- Thème noir & blanc clair/sombre

## Lancer

```bash
cd mini-projects/flutter/fitcoach_pro
flutter pub get
flutter run -d web-server --web-hostname=localhost --web-port=5553
```

Ouvrir http://localhost:5553
