# Calcify — Calculatrice

Calculatrice Flutter autonome : opérations de base, √, x², %, historique.

## Fonctionnalités

- Addition, soustraction, multiplication, division
- √, x², %, ±, décimal
- Historique réutilisable
- Thème noir & blanc clair/sombre

## Lancer

```bash
cd mini-projects/flutter/calcify_app
flutter pub get
flutter run -d web-server --web-hostname=localhost --web-port=5548
```

Ouvrir http://localhost:5548
