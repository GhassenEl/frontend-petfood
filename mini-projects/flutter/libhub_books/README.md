# LibHub — Bibliothèque en ligne

App Flutter : livres, journaux, Coran, hadith, emprunts numériques.

## Fonctionnalités

- **Accueil** : collections, KPIs, emprunts en cours
- **Livres** : romans, sciences, histoire, tech
- **Journaux** : revues académiques, presse
- **Coran** : Coran, tafsir, hadith, spiritualité
- **Emprunts** : gestion membres, retours, statuts
- Tap = emprunter · appui long = retour
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5539
```

Ouvrir http://localhost:5539
