# SummerStay — Location maisons d'été, bungalows & villas

Application Flutter autonome pour la location saisonnière (maisons d'été, bungalows, villas) avec assistant **SummerBot IA**.

## Fonctionnalités

- Catalogue biens (villa / bungalow / maison d'été)
- Filtres type + recherche ville/région
- Réservations dynamiques (CRUD statut)
- KPIs : disponibles, villas, CA
- **SummerBot** : budget, piscine, vue mer, famille, villes tunisiennes
- Thème noir & blanc clair/sombre

## Lancer

```bash
cd mini-projects/flutter/summerstay_rentals
flutter pub get
flutter run -d web-server --web-hostname=localhost --web-port=5546
```

Ouvrir http://localhost:5546
