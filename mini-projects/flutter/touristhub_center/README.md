# TouristHub — Centre de tourisme

App Flutter autonome : circuits touristiques, clients, réservations et guides.

## Fonctionnalités

- **Circuits** : Sidi Bou Said, désert, patrimoine… (région, durée, prix)
- **Clients** : touristes locaux et internationaux
- **Réservations** : participants, statuts cliquables
- **Guides** : langues parlées, charge hebdo
- Thème noir & blanc · bouton **+** sur chaque onglet

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5529
```

Ouvrir http://localhost:5529
