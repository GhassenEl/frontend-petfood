# TaxiGo — Application taxi type Uber

App Flutter mobile/web : gestion de courses, chauffeurs et tarifs.

## Fonctionnalités

- **Dashboard** : courses actives, CA, chauffeurs en ligne
- **Chauffeurs** : statut (Disponible / En course / Hors ligne), note, véhicule
- **Courses** : départ → destination, assignation chauffeur, statuts cliquables
- **Tarifs** : zones et grilles kilométriques
- Thème noir & blanc · bouton **+** fonctionnel sur chaque onglet

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5531
```

Ouvrir http://localhost:5531
