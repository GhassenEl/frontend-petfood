# MatchUp — Matchs amicaux Football & Basket

App Flutter pour organiser des matchs amicaux entre amis ou équipes.

## Fonctionnalités

- **Dashboard** : matchs ouverts, joueurs inscrits
- **Football** : créer matchs, inscriptions (appui long), statuts
- **Basket** : idem pour le basketball
- **Équipes** : capitaines, membres, victoires
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5536
```

Ouvrir http://localhost:5536
