# RideShop — Trottinettes & vélos

App Flutter mobile/web : boutique trottinettes et bicyclettes.

## Fonctionnalités

- **Catalogue** : trottinettes électriques, vélos ville, VTT, modèles enfant
- **Commandes** : statuts cliquables, CA en temps réel
- **Stock** : tap sur un produit pour décrémenter le stock
- **Promotions** : codes promo
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5530
```

Ouvrir http://localhost:5530
