# StayHub — Réservation hôtels

App Flutter : réserver des hôtels en Tunisie et gérer les réservations.

## Fonctionnalités

- **Accueil** : KPIs, chambres disponibles, CA
- **Hôtels** : catalogue, étoiles, prix/nuit — tap pour réserver
- **Réservations** : check-in/out, statuts cliquables
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5540
```

Ouvrir http://localhost:5540
