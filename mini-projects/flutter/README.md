# Mini-projects Flutter

Chaque application Flutter a sa **propre branche** dédiée (projet isolé à la racine).
Elles ne sont plus versionnées sous la branche frontend (`mini-projects` / `main`).

## Cloner un projet

```bash
git clone -b project-calcify --single-branch https://github.com/GhassenEl/frontend-petfood.git calcify_app
cd calcify_app
flutter pub get
flutter run -d chrome
```
