# Carthage Land — Parc d'attractions

App Flutter : parc d'attractions Carthage Land (attractions, billets, spectacles).

## Fonctionnalités

- **Accueil** : attractions ouvertes, billets, CA, spectacles
- **Attractions** : zones, âge min, temps d'attente — tap pour changer le statut
- **Billets** : journée, scolaire, premium — statuts cliquables
- **Spectacles** : parade, pirate, feu d'artifice — appui long pour réserver une place
- Thème noir & blanc · pied de page visible · FAB +

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5542
```

Ouvrir http://localhost:5542
