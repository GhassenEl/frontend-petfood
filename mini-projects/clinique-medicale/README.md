# MediLink — Clinique médicale

Mini-projet autonome : gestion médecins, patients, rendez-vous et ordonnances.

## Fonctionnalités dynamiques

- Ajout / suppression médecins, patients, RDV, ordonnances
- Statut RDV cliquable (Planifié → En salle → Terminé → Annulé)
- KPIs calculés en temps réel
- Sauvegarde `localStorage`
- Thème noir & blanc

## Lancer

```bash
npx --yes serve -l 5524
```

Ouvrir http://localhost:5524
