# WildZoo — Zoo animaux

App Flutter : gestion d'un zoo (animaux, habitats, billets de visite).

## Fonctionnalités

- **Accueil** : animaux visibles, habitats, CA billets
- **Animaux** : espèces, habitats, statut (Visible / Soins / Quarantaine)
- **Habitats** : savane, aquarium, volière — capacité
- **Visites** : billets famille / groupe / adulte — validation
- Thème noir & blanc · pied de page visible · FAB +

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5543
```

Ouvrir http://localhost:5543
