# LexHub — Cabinet d'avocats + IA

Application Flutter : avocats, RDV clients, audiences tribunal, secrétariat et assistant LexBot.

## Fonctionnalités

- **Accueil** : KPIs, RDV du jour, tâches secrétariat urgentes
- **Avocats** : spécialités, barreau, statuts (Disponible / En audience)
- **RDV** : consultations, confirmations, annulations
- **Tribunal** : audiences programmées, reports, affaires
- **Secrétariat** : tâches, priorités, assignation
- **LexBot IA** : agenda, audiences, avocats dispo, aide secrétariat
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5538
```

Ouvrir http://localhost:5538
