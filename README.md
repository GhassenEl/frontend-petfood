# BankFlow — Application bancaire

App Flutter mobile/web : gestion bancaire personnelle et professionnelle.

## Fonctionnalités

- **Accueil** : solde total, revenus/dépenses, dernières opérations
- **Comptes** : courant, épargne, pro — IBAN et soldes
- **Opérations** : virements, retraits, crédits — statuts cliquables
- **Cartes** : Visa, Mastercard, virtuelles — blocage/déblocage
- Virement rapide depuis l'accueil
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5537
```

Ouvrir http://localhost:5537
