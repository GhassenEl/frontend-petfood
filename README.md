# FactoryLink — Usine connectée

Application Flutter IoT pour usine connectée : lignes de production, machines, capteurs, alertes, maintenance et FactoryBot IA.

## Fonctionnalités

- **Dashboard** — OEE, production, énergie, alertes
- **Lignes** — cadence, démarrage/arrêt, filtres
- **Machines IoT** — température, vibration, puissance, QR
- **Alertes** — vibration, arrêt, température, maintenance
- **Maintenance** — ordres de travail planifiés
- **FactoryBot IA** — OEE, lignes, capteurs, maintenance
- Thème noir & blanc clair/sombre

## Lancer

```bash
cd mini-projects/flutter/usine_connectee
flutter pub get
flutter run -d web-server --web-hostname=0.0.0.0 --web-port=5555
```

Ouvrir http://localhost:5555
