# GreenLeaf — Boutique plantes en ligne + IA

App Flutter : vente de plantes en ligne avec assistant GreenBot.

## Fonctionnalités

- **Boutique** : KPIs, suggestions IA, commandes en cours
- **Catalogue** : plantes, prix, stock, conseils d'entretien (tap)
- **Commandes** : statuts cliquables, livraison en ligne
- **GreenBot IA** : recommandations (débutant, appartement, soleil), arrosage, analyse panier
- Panier dynamique · thème noir & blanc

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5534
```

Ouvrir http://localhost:5534
