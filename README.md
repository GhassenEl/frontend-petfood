# KidsCircle — Jardin d'enfants & crèche

App Flutter : gestion de crèche avec enfants, activités, notifications, historique et KidsBot IA.

## Fonctionnalités

- **Accueil** : présences, activités du jour, alertes
- **Enfants** : groupes (Bébé/Petits/Moyens/Grands), allergies, statut cliquable
- **Activités** : ateliers créatifs/sport/éveil — inscription (appui long)
- **Notifications** : repas, santé, présence — badge non lues
- **Historique** : arrivées, absences, siestes, repas
- **KidsBot IA** : présences, allergies, idées d'atelier, alertes
- Thème noir & blanc · pied de page visible · FAB +

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5545
```

Ouvrir http://localhost:5545
