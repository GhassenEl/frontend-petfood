# TrailHub — Randonnées & excursions

App Flutter pour organiser randonnées et excursions touristiques.

## Fonctionnalités

- **Accueil** : activités ouvertes, participants
- **Randonnées** : difficulté, durée, guide — inscription (appui long)
- **Excursions** : destinations, prix, places — inscription (appui long)
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5541
```

Ouvrir http://localhost:5541
