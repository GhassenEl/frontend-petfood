# LinguaHub — Centre de langues (Flutter)

App Flutter autonome : cours de langues, étudiants, planning et formateurs.

## Fonctionnalités

- **Cours** : langues, niveaux (A1–C2), durée, prix, mode (présentiel/hybride/en ligne)
- **Étudiants** : inscription, progression (%), statuts
- **Planning** : séances du jour avec formateur et salle
- **Formateurs** : langues enseignées, charge hebdo
- Thème noir & blanc · bouton **+** sur chaque onglet

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5526
```

Ouvrir http://localhost:5526
