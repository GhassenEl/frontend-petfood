# UniCampus — Éducation universitaire en ligne

Plateforme Flutter : professeurs et étudiants universitaires.

## Fonctionnalités

- **Dashboard** : profs en ligne, étudiants actifs, sessions live
- **Professeurs** : matières, départements, statut (En ligne / Hors ligne)
- **Étudiants** : filières, niveaux, moyennes
- **Sessions** : cours visio, inscriptions, statuts cliquables
- Thème noir & blanc · bouton **+** fonctionnel

## Lancer

```bash
flutter pub get
flutter run -d web-server --web-port=5535
```

Ouvrir http://localhost:5535
