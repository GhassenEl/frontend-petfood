🐾 PetfoodTN — Accès plateforme

Ouvrez ce fichier hors Cursor (double-clic). Démarrez d’abord les serveurs avec scripts\windows\4-DEMARRER-TOUT.bat.

⚠️ Les liens localhost fonctionnent seulement si les serveurs tournent sur votre PC. Voir ACCES_HORS_CURSOR.md pour installation et téléchargement GitHub.

🌐 Web — Frontend + Backend

Plateforme web Ouvrir :3001 http://localhost:3001
Démo jury Ouvrir jury-demo http://localhost:3001/jury-demo
Connexion Login http://localhost:3001/login
API Backend Health :5002 http://localhost:5002/api

🧠 FastAPI — ML (port 8000)

Service ML Ouvrir :8000 http://localhost:8000
Swagger / Docs /docs http://localhost:8000/docs
Santé ML /health http://localhost:8000/health

📱 Flutter — App mobile

Flutter Web Ouvrir :8080 http://localhost:8080

Lancer : mobile_app\start.bat ou scripts\windows\3-DEMARRER-FLUTTER-WEB.bat
APK release : mobile_app\build\app\outputs\flutter-apk\app-release.apk
